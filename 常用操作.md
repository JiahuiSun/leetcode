- vector初始化

```C++
vector<int> nums(10, 0); // 长度为10, 值为0
vector<int> nums = {1, 2, 3};
vector<int> nums {1, 2, 3};
```



- 遍历vector

```C++
// 索引
for(int i=0; i<nums.size(); i++)
	cout << nums[i];

// auto
for(auto &p : nums)
    cout << p;

// 迭代器
for(auto p=nums.cbegin(); p!=nums.cend(); ++p)
	cout << *p;
```
